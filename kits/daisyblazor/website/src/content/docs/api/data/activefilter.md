Tag rendered above the filter panel for each currently-applied filter.

## Properties

### `Key`

Stable identifier so the panel can ask the caller to remove this specific filter.

### `Label`

Localized text displayed inside the chip.

## Methods

### `ActiveFilter(string, string)`

Tag rendered above the filter panel for each currently-applied filter.

- `Key` — Stable identifier so the panel can ask the caller to remove this specific filter.
- `Label` — Localized text displayed inside the chip.
