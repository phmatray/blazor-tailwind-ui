## Properties

### `Grid`

Owning grid, supplied via cascading value.

### `InfoSeparatorText`

Separator word in the range text (e.g. "of").

### `PageSizeOptions`

Page-size options. When set, mirrored onto the grid.

### `RowsPerPageText`

Label shown before the page-size select.
