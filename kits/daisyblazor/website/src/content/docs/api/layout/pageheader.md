Page-level header row: title + optional subtitle on the left, optional action slot on the right.

## Properties

### `Actions`

Optional right-aligned content (typically action buttons).

### `Class`

Optional CSS class forwarded to the root container.

### `Style`

Optional inline style forwarded to the root container.

### `Subtitle`

Optional secondary line beneath the title.

### `Title`

Main page title rendered as `Typo.h4`.

### `UserAttributes`

Unmatched attributes splatted onto the root container.
