## Properties

### `Area`

Fill the area beneath the line.

### `Color`

Optional explicit CSS color; defaults to the primary theme token.

### `Data`

The values to plot, oldest-to-newest.

### `Height`

CSS height (inline; defaults to a compact cell size).

### `ShowEndDot`

Draw a dot on the most recent value.

### `StrokeWidth`

Line stroke width in user units.

### `Title`

Accessible description; defaults to a generic sparkline label.

### `Width`

CSS width (inline; defaults to a compact cell size).
