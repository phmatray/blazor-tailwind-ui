## Properties

### `Area`

Fill the area under each line.

### `Height`

CSS height of the SVG plot area.

### `Labels`

X-axis category labels (index-aligned with each series' values).

### `Series`

One or more named series to plot.

### `ShowGrid`

Draw horizontal gridlines and value labels.

### `ShowLegend`

Show a legend when more than one series is present.

### `ShowPoints`

Draw a marker at each data point.

### `Smooth`

Smooth the line with cubic curves instead of straight segments.

### `Title`

Optional heading rendered above the chart.

### `Width`

CSS width of the chart wrapper.
