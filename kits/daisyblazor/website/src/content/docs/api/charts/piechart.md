## Properties

### `CenterLabel`

Optional label rendered in the center (donut only).

### `Data`

Slice data; each point's value contributes to the whole.

### `Height`

CSS height of the SVG.

### `InnerRadiusRatio`

Inner radius as a fraction of the outer radius (0 = pie). Overridden by donut subclass.

### `ShowLegend`

Show a legend with labels and percentages.

### `Title`

Optional heading rendered above the chart.

### `Width`

CSS width of the chart wrapper.
