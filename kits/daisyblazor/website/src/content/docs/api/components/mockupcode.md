## Properties

### `ChildContent`

Content rendered verbatim inside the code block (use when you need custom <pre> rows).

### `Lines`

Convenience list of code lines; each is wrapped in `<pre data-prefix="..."><code>` automatically.

### `Prefix`

The prefix shown before each line when using `MockupCode.Lines`. Defaults to "$".
