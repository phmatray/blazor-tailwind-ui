## Properties

### `Active`

Whether the swap is in the active (on) state.

### `ActiveChanged`

Raised when the active state changes.

### `Animation`

Animation applied to the swap transition.

### `Off`

Content shown when the swap is inactive (unchecked).

### `On`

Content shown when the swap is active (checked).
