## Properties

### `BackgroundImage`

URL of a background image. When provided a semi-transparent
`hero-overlay` layer is added and the image is applied as an
inline `background-image` style on the root element.

### `ChildContent`

Content rendered inside `hero-content`.

### `ContentClass`

Extra CSS classes forwarded to the inner `hero-content` wrapper.

### `MinHeight`

Tailwind min-height utility applied to the root element.
Defaults to `"min-h-screen"`. Pass e.g. `"min-h-96"` for a shorter hero.
