## Methods

### `ResetAsync()`

MudBlazor compat: reset the form's validation state. No-op.

### `ResetValidation()`

MudBlazor compat: reset validation only. No-op.

### `Validate()`

MudBlazor compat: trigger validation. No-op (returns current validity).
