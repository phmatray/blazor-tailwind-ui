## Properties

### `Activator`

The trigger element rendered inside the <summary> tag.

### `ChildContent`

The dropdown panel content (typically a <ul> menu list).

### `ContentClass`

Extra CSS classes forwarded to the.dropdown-content wrapper.

### `Hover`

When true, adds.dropdown-hover so the panel opens on mouse-over (CSS only).

### `Open`

Forces the dropdown open via the HTML open attribute and.dropdown-open class.

### `Placement`

Placement of the dropdown content relative to the trigger.
