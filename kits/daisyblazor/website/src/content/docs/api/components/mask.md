## Properties

### `ChildContent`

Content to clip — typically an `<img>` element.

### `Half`

Optional half-mask modifier. Pass `1` to apply `mask-half-1`
(left/top half) or `2` for `mask-half-2` (right/bottom half).
Any other value is ignored.

### `Shape`

Geometric shape used to clip the content.
