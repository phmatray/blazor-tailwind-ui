## Properties

### `ActivatorContent`

Custom activator markup; receives a context with `ToggleAsync`.

### `Icon`

Material Symbols ligature for the default icon activator.

### `Label`

Text label for the default button activator.

## Methods

### `NotifyItemClickedAsync()`

Called by child MenuItems after activation to dismiss the menu.
