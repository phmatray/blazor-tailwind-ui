## Properties

### `Animation`

Accepted for MudBlazor compatibility; daisyUI handles the pulse animation.

### `Height`

CSS height (e.g. "1rem", "40px"); applied as inline style.

### `SkeletonType`

Placeholder shape (Circle renders a rounded block).

### `Width`

CSS width (e.g. "100%", "200px"); applied as inline style.
