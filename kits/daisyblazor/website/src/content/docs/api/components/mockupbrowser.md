## Properties

### `BorderColor`

Optional border color token applied as `border-{token}`; defaults to no extra border class.

### `ChildContent`

Content rendered inside the browser display area.

### `Toolbar`

Optional custom toolbar content; rendered inside the toolbar bar instead of the default URL input.

### `Url`

Optional URL string shown in the address bar input.
