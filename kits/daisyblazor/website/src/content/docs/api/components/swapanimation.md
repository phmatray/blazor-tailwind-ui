Animation modifier applied to the daisyUI swap component.

## Fields

### `Flip`

Applies the `swap-flip` class for a flip transition.

### `None`

No animation; the swap content changes without transition.

### `Rotate`

Applies the `swap-rotate` class for a rotation transition.
