---
title: "ConfirmDialog"
description: "Standard confirm/cancel dialog."
---

Standard confirm/cancel dialog. Pass `ConfirmDialog.CancelText` for a localized cancel label.

## Properties

### `ButtonText`

Label of the confirm button.

### `CancelText`

Label of the cancel button. Defaults to `"Cancel"`; pass a localized string to override.

### `Color`

Color applied to the confirm button.

### `ContentText`

Body text shown inside the dialog.
