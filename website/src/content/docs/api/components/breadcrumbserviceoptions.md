Options for the daisyUI breadcrumb adapter.

## Properties

### `HomeIcon`

Icon for the Home breadcrumb. Defaults to `Icons.Material.Filled.Home`.

### `HomeTitleKey`

Resource key used to localize the Home breadcrumb. Defaults to `"Breadcrumb_Home"`.
