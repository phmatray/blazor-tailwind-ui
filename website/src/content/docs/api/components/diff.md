## Properties

### `AspectClass`

Tailwind aspect-ratio utility applied to the root element (e.g. "aspect-16/9", "aspect-4/3", "aspect-square"). Defaults to "aspect-16/9".

### `Item1`

Content rendered in the first (left/bottom) diff pane.

### `Item2`

Content rendered in the second (right/top) diff pane.
