## Properties

### `ChildContent`

The message content rendered inside the.chat-bubble.

### `Color`

Semantic color applied to the bubble via.chat-bubble-{token}.

### `Footer`

Optional text rendered in the.chat-footer slot (e.g. "Delivered", "Seen").

### `Header`

Optional text rendered in the.chat-header slot (e.g. sender name + timestamp).

### `Image`

Optional avatar content rendered in the.chat-image slot.

### `Placement`

Whether the bubble appears on the left (Start) or right (End) side.
