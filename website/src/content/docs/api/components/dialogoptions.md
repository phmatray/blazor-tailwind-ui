MudBlazor-compatible display options for a dialog.

## Properties

### `BackdropClick`

Allow dismissing the dialog by clicking the backdrop.

### `CloseButton`

Show an explicit close (x) button in the header.

### `CloseOnEscapeKey`

Allow dismissing the dialog with the Escape key.

### `FullWidth`

Stretch the dialog box to the chosen max width.

### `MaxWidth`

Maximum width of the dialog box.

### `NoHeader`

Center the dialog vertically (no-op placeholder for parity).
