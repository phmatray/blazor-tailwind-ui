Per-message snackbar configuration, mirroring the MudBlazor options surface used across DaisyBlazor.

## Properties

### `PreventDuplicates`

When true, an identical message will not be added again while it is still shown.

### `ShowCloseIcon`

Show a manual close button on the toast.

### `VisibleStateDuration`

How long (ms) the toast stays visible before it auto-dismisses.
