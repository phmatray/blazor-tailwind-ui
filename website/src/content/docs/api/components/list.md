## Properties

### `Background`

Apply bg-base-100 background (matches the daisyUI list demo surface).

### `ChildContent`

The row content — typically one or more `ListRow` components.

### `Rounded`

Apply rounded-box border-radius around the list.

### `Shadowed`

Apply a drop shadow via the daisyUI shadow utility.
