## Properties

### `Center`

Adds.footer-center to centre-align all items.

### `ChildContent`

Content of the footer, typically one or more <nav> sections.

### `Color`

daisyUI / Tailwind semantic colour for the footer background (e.g. Color.Neutral).

### `Horizontal`

When `true` adds.footer-horizontal; when `false` adds.footer-vertical;
when `null` (default) neither modifier is applied and daisyUI uses its default
responsive direction.

### `Padding`

Padding utility class applied to the footer. Defaults to "p-10".
