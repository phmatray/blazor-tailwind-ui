## Properties

### `Animated`

When `true` renders a ping / pulse animation (inline-grid + animate-ping layer).

### `AriaLabel`

Accessible label surfaced via `aria-label` (recommended when the dot conveys status without visible text).

### `Color`

Semantic color of the dot; maps to daisyUI `status-{token}`.

### `Size`

Size of the dot; maps to daisyUI `status-{size}`.
