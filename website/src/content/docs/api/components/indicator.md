## Properties

### `ChildContent`

The base element over which the indicator is placed.

### `Horizontal`

Horizontal position of the indicator item (Start / Center / End).

### `Item`

The element placed as the indicator overlay (badge, dot, icon, etc.).

### `Vertical`

Vertical position of the indicator item (Top / Middle / Bottom).
