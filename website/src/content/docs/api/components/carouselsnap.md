Scroll-snap alignment applied to a daisyUI carousel.

## Fields

### `Center`

Snaps items to the centre of the scroll container (`carousel-center`).

### `End`

Snaps items to the end of the scroll container (`carousel-end`).

### `None`

No explicit snap class; browser default snap behaviour.
