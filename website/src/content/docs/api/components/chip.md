## Properties

### `CloseIcon`

Close icon ligature for the optional close button.

### `Icon`

Leading icon ligature.

### `OnClose`

Raised when the close button is clicked; rendering it requires a delegate.

### `Text`

Text shown when no ChildContent is supplied.

### `Value`

Backing value for selection scenarios.
