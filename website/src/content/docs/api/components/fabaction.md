## Properties

### `ChildContent`

Arbitrary child content (replaces the auto-rendered Icon when set).

### `Color`

Semantic color applied to the action button.

### `Icon`

Material Symbols ligature shown inside the action button.

### `OnClick`

Click callback.

### `Tooltip`

Optional tooltip text displayed next to the button.

### `TooltipPosition`

Tooltip placement relative to the button (left / right / top / bottom).
