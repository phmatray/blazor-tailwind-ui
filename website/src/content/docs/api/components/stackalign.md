---

Alignment modifier for the daisyUI `stack` utility on a `StackOverlay` component.

## Fields

### `Bottom`

Align children to the bottom (`stack-bottom`).

### `End`

Align children to the end (right in LTR) (`stack-end`).

### `Start`

Align children to the start (left in LTR) (`stack-start`).

### `Top`

Align children to the top (`stack-top`).
