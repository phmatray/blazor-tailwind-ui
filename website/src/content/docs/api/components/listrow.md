## Properties

### `ChildContent`

Freeform content rendered directly inside the row.
Used when Icon/Title/Subtitle are not set.

### `Icon`

Material Symbols ligature name for the leading icon (e.g. Icons.Material.Filled.Star).

### `Subtitle`

Secondary / caption text shown beneath the title.

### `Title`

Primary text displayed in the grow column.

### `TrailingContent`

Optional trailing slot rendered in a list-col-wrap column on the right.
Only used together with the Icon/Title/Subtitle convenience layout.
