## Properties

### `ChildContent`

Speed-dial action items (typically <FabAction> elements or plain daisyUI buttons).

### `Color`

Semantic color applied to the trigger button.

### `Flower`

When true the items spread in a flower/radial pattern (.fab-flower).
When false (default) they stack in a vertical list.

### `Icon`

Material Symbols ligature for the trigger icon. Defaults to "add".

### `ShowClose`

When true a.fab-close layer is rendered so the user can click the trigger again
(or anywhere outside) to collapse the speed dial.
