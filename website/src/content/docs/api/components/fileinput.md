## Properties

### `Accept`

Constrains the file picker to specific MIME types or extensions (e.g. "image/*", ".pdf").

### `Color`

Semantic color applied as file-input-{token} (e.g. file-input-primary).

### `Ghost`

Applies the ghost variant (file-input-ghost) for a borderless, transparent style.

### `Multiple`

When true the browser allows picking more than one file.

### `OnChange`

Raised when the user selects one or more files.

### `Size`

Size of the control: Small → file-input-sm, Medium → file-input-md, Large → file-input-lg.
