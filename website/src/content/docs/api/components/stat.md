## Properties

### `Actions`

Optional action content (e.g. buttons or links) rendered below the description.

### `Desc`

Supplementary description shown below the value.

### `DescColor`

Semantic color applied to the stat-desc text (maps to text-{token}).

### `Figure`

Optional figure content (e.g. an icon or avatar) rendered in stat-figure.

### `Title`

Stat label shown above the value (e.g. "Total Flights").

### `Value`

Primary numeric or text value displayed prominently.
