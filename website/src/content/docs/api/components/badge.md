## Properties

### `Content`

Badge content (number or text). Capped by `Badge.Max` when numeric.

### `Dot`

Render a small dot with no content.

### `Max`

Maximum numeric value before showing "{Max}+".

### `Overlap`

Overlap the badge onto the child (kept for MudBlazor compatibility).
