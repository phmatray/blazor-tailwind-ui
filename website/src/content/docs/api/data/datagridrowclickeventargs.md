---

Event arguments for a grid row click. Mirrors MudBlazor's `DataGridRowClickEventArgs<T>`.

## Type parameters

- `T` — Row item type.

## Properties

### `Item`

The clicked row item.

### `RowIndex`

Zero-based index of the clicked row within the current page.
