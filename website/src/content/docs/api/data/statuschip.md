A filled `Chip<T>` variant for status indicators (e.g. flight status, report state).

## Properties

### `Class`

Additional CSS classes applied to the chip.

### `Color`

Color applied to the chip.

### `Icon`

Optional leading icon (use values from `Icons`).

### `Label`

Text displayed inside the chip.

### `Size`

Chip size; defaults to `Size.Small`.

### `Style`

Inline style applied to the chip.

### `UserAttributes`

Arbitrary attributes splatted onto the chip.
