Color-bordered KPI tile: caption + large value + optional subtitle.

## Properties

### `Class`

Additional CSS classes applied to the tile.

### `Color`

CSS color used for the left border and the value text. Defaults to a light blue.

### `Label`

Caption shown above the value. Already-resolved text (pass `@L["…"]` at the call site to localize).

### `Style`

Inline style appended to the tile.

### `Subtitle`

Optional smaller caption beneath the value.

### `UserAttributes`

Arbitrary attributes splatted onto the tile.

### `Value`

Main metric value.
