## Properties

### `Data`

Flat data points; used when no `BarChart.Series` are supplied.

### `Height`

CSS height of the SVG plot area.

### `Horizontal`

Render bars horizontally instead of vertically.

### `Labels`

Category labels for grouped `BarChart.Series` mode (index-aligned with series values).

### `Series`

Grouped series (one bar per series within each category). Takes precedence over `BarChart.Data`.

### `ShowGrid`

Draw value gridlines and labels.

### `ShowLegend`

Show a legend when more than one series is present.

### `Title`

Optional heading rendered above the chart.

### `Width`

CSS width of the chart wrapper.
