Paper-wrapped detail card with an icon+title header and a dense, hoverable
daisyUI table body. Pair with `DetailRow`.

## Properties

### `ChildContent`

Body content — typically a sequence of `DetailRow` elements.

### `Class`

Optional CSS class forwarded to the root paper.

### `Dense`

When true, applies dense styling to the inner table. Defaults to true.

### `Elevation`

Paper elevation. Defaults to 1.

### `Icon`

Optional icon shown to the left of `DetailCard.Title`.

### `Style`

Optional inline style forwarded to the root paper.

### `Title`

Card title shown in the header row.

### `UserAttributes`

Unmatched attributes splatted onto the root paper.
