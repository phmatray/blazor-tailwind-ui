Standard detail-page scaffold: back link, title, optional status slot, main content,
optional sidebar, optional related-content area.

## Properties

### `BackLabel`

Label of the back button. Defaults to `"Back"`; pass a localized string to override.

### `BackRoute`

Route to navigate to when the back button is clicked.

### `Class`

Optional CSS class forwarded to the root container.

### `MainContent`

Primary content of the detail page.

### `RelatedContent`

Optional full-width section rendered below the main grid.

### `Sidebar`

Optional sidebar (right column on md+ viewports).

### `StatusSlot`

Optional element rendered next to the title (typically a status chip).

### `Style`

Optional inline style forwarded to the root container.

### `Subtitle`

Optional secondary line beneath the title.

### `Title`

Page title.

### `UserAttributes`

Unmatched attributes splatted onto the root container.
